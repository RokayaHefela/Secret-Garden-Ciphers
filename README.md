# 🔐 Secret Garden Ciphers

A web-based cryptography application developed as a **team project** for the **Cryptography** course.

The application provides an interactive interface for encrypting and decrypting text using multiple classical and modern cryptographic algorithms.

---

## 🌐 Live Demo

🔗 [https://rokayahefela.github.io/Secret-Garden-Ciphers/](https://rokayahefela.github.io/Secret-Garden-Ciphers/)

---

## ✨ Features

- Encrypt and decrypt text
- Interactive and user-friendly interface
- Multiple cryptographic algorithms
- Educational demonstration of encryption techniques

---

## 🔒 Implemented Algorithms

- Caesar Cipher
- Vigenère Cipher
- Hill Cipher
- Affine Cipher
- Rail Fence Cipher
- Columnar Transposition Cipher
- One-Time Pad (OTP)
- RSA
- Diffie–Hellman Key Exchange
- AES
- DES
- Hash Functions
