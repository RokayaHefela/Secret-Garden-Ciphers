✨ Secret Garden Ciphers ✨

🌸 Caesar Cipher

Shift each letter by a fixed number of positions in the alphabet.

➜

🌺 Vigenere Cipher

Uses a keyword to shift letters with varying amounts.

➜

🌷 Hill Cipher

Matrix-based encryption using a 2x2 key matrix. Enter 4 numbers for the matrix.

💡 Make sure determinant is coprime with 26 (valid example: 3,3,2,5)

➜

🌹 Affine Cipher

Uses mathematical function (ax + b) mod 26. 'a' must be coprime with 26.

➜

🌻 One Time Pad

The unbreakable cipher! Key must be random and at least as long as the message.

➜

🌼 Substitution Cipher

Replace each letter with another from a shuffled alphabet.

💡 Default is reverse alphabet. Click Random for a shuffled key!

➜

🚂 Rail Fence Cipher

Write the message in zig-zag rows and read off by rows.

➜

📋 Columnar Transposition

Write letters in columns using a keyword, then read by sorted column order.

➜

🔑 RSA Cipher

Public-key encryption using two prime numbers.

➜

🤝 Diffie-Hellman

Agree on a shared secret by exchanging public values over a prime field.

➜

🔐 AES Cipher

Modern symmetric encryption using AES-CBC.

➜

🛡️ DES Cipher

Simple block cipher demonstration using 8-byte key and Feistel rounds.

➜

🔗 Hash Function

Compute a digest of your message with SHA-256 or SHA-512.
